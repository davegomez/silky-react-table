silky-react-table is a small and simple React Component library that displays tabulated data in an HTML table using the latest HTML standards, and adds accessibility features that every website or web application should include these days.

Smart defaults

Smart defaults allow you to use silky-react-table to display tabulated data without any configuration; creating a simple, accessible, and semantically correct HTML table.

Accessible

Accessibility is a key part of silky-react-table since not all your visitors and customers will consume your data in the same way, so you can have your data accessible by using features like caption and scope by default in your tables.

Customizable

When passing an optional options object, you can customize the way it will display and format your data in several ways, such as adding headers and footers; use default or custom sorting methods; and even retrieve and format your data before rendering by using transformation functions.

Blank visual state

By default silky-react-table doesn't includes any style or visual modification but all the markup is under the class name silky-react-table, you can use this class to safely apply any visual style you want without adding additional specificity to your CSS code.
